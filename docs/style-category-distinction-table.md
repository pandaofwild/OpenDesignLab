# OpenDesignLab Style Category Distinction Table

This table is the working comparison sheet for keeping styles inside the same category visually separable. Moodboard signals are translated into surface, border, texture, spacing, and media treatments. Reference-site signals are translated into web information structure, navigation rhythm, component grammar, and section order.
